# 🔥 DarkXNetwork Advanced Arsenal v1.0
**Developer:** Ashif 

Welcome to DarkXNetwork, an advanced multi-tool arsenal for Termux.

### ⚙️ Installation Guide for Termux
Open your Termux and copy-paste these commands one by one:

1. `git clone https://github.com/aishorts7083-code/DarkXNetwork.git`
2. `cd DarkXNetwork`
3. `chmod +x install.sh`
4. `./install.sh`

### 🚀 How to Run
After installation, you don't need to use python commands. Just type this anywhere in your Termux:

`darkxnetwork`
